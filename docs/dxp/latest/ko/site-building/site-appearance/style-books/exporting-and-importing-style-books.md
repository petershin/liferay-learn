# 스타일북 내보내기 및 가져오기

스타일 북을 내보내고 가져와서 Liferay 설치 간에 [ZIP 파일](#exporting-style-books-as-a-zip-file) 또는 [Liferay Archive(LAR) 파일](#exporting-style-books-as-a-lar-file) 으로 공유할 수 있습니다.

## 스타일 북을 ZIP 파일로 내보내기

1. **사이트 메뉴**( ![Site Menu icon](../../../images/icon-product-menu.png) )을 열고 **디자인** &rarr; **스타일 북** 로 이동합니다.

1. 내보내려는 스타일 북을 찾아 **작업**( ![Actions Icon](../../../images/icon-actions.png) ) &rarr; **내보내기** 클릭합니다.

1. ZIP 파일이 컴퓨터에 다운로드됩니다. 여기에는 두 개의 파일이 포함되어 있습니다.

   `style-book.json` 스타일북의 이름과 `frontend-token-values.json`에 대한 경로가 포함되어 있습니다.

   `frontend-token-values.json` 스타일 북의 토큰 값 목록이 포함되어 있습니다. 각 토큰은 CSS 변수 이름과 이에 매핑된 값을 참조합니다.

이제 ZIP 파일을 공유하고 가져올 수 있습니다.

## ZIP 파일에서 스타일 북 가져오기

1. **사이트 메뉴**( ![Site Menu icon](../../../images/icon-product-menu.png) )을 열고 **디자인** &rarr; **스타일 북** 로 이동합니다.

1. 응용프로그램 막대에서 **옵션**( ![Options icon](../../../images/icon-options.png) ) &rarr; **가져오기** 클릭합니다.

1. 가져오려는 ZIP 파일을 선택하세요.

   동일한 이름의 기존 스타일 북을 덮어쓰려면 **기존 스타일 북 덮어쓰기** 선택합니다. 옵션을 선택하지 않은 상태에서 기존 스타일북과 동일한 이름의 스타일북을 가져오면 오류 메시지가 표시됩니다.

1. **가져오기** 클릭합니다.

이제 가져온 스타일 가이드를 사용할 수 있습니다.

## 스타일 북을 LAR 파일로 내보내기

1. **사이트 메뉴**( ![Site Menu icon](../../../images/icon-product-menu.png) )을 열고 **디자인** &rarr; **스타일 북** 로 이동합니다.

1. 내보내려는 스타일 북을 선택합니다. 응용프로그램 막대에서 **옵션**( ![Options Icon](../../../images/icon-options.png) ) &rarr; **내보내기/가져오기** 클릭합니다.

   ![Style Book을 내보내려면 Style Books 앱의 옵션으로 이동하세요.](./exporting-and-importing-style-books/images/01.png)

1. 내보내기/가져오기 모달에서 **내보내기** 선택합니다.

1. **새 내보내기 프로세스를 시작합니다** .

   ```{note}
   *현재 및 이전* 탭을 선택하여 이전 프로세스 목록을 보고 내보낸 LAR 파일을 다운로드할 수 있습니다.
   ```

1. 새 내보내기 프로세스 [시작할 때 LAR 파일 이름을 지정하고 프로세스를 구성할 수 있습니다](#configuring-export-and-import-processes-as-a-lar-file) .

   ![스타일 북 내보내기 또는 가져오기를 선택하고 새 프로세스를 시작하거나 현재 및 이전 프로세스를 확인할 수 있습니다.](./exporting-and-importing-style-books/images/02.png)

1. 내보내기 프로세스를 시작하려면 **내보내기** 클릭하세요.

이제 LAR 파일을 공유하고 가져올 수 있습니다.

## 스타일 북을 LAR 파일로 가져오기

1. **사이트 메뉴**( ![Site Menu icon](../../../images/icon-product-menu.png) )을 열고 **디자인** &rarr; **스타일 북** 로 이동합니다.

1. 가져오려는 스타일 북을 선택합니다. 응용프로그램 막대에서 **옵션**( ![Options Icon](../../../images/icon-options.png) ) &rarr; **내보내기/가져오기** 클릭합니다.

1. 내보내기/가져오기 모달에서 **가져오기** 선택합니다.

1. **새 가져오기 프로세스를 시작합니다** .

   ```{note}
   *현재 및 이전* 탭을 선택하여 이전 프로세스 목록을 볼 수 있습니다.
   ```

1. LAR 파일을 업로드하고 **계속** 클릭합니다.

1. [프로세스를 구성하십시오](#configuring-export-and-import-processes-as-a-lar-file) .

1. 가져오기 프로세스를 시작하려면 **가져오기** 클릭하세요.

이제 가져온 스타일 북을 사용할 수 있습니다.

## 내보내기 및 가져오기 프로세스를 LAR 파일로 구성

Liferay는 내보내기 및 가져오기 프로세스 구성을 위한 설정을 제공합니다.

| 환경                          | 묘사                                                                                                                                                                                                                                                                                                                                   |
|:--------------------------- |:------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 신청                          | 선택하면 스타일 북의 구성 설정을 LAR 파일로 내보내거나 가져옵니다.                                                                                                                                                                                                                                                                                              |
| 컨텐츠                         | 내보내려는 콘텐츠의 날짜 범위를 선택합니다. 지정된 날짜 범위 내에 스타일 북에 추가된 콘텐츠가 LAR 파일에 포함됩니다.                                                                                                                                                                                                                                                                 |
| 삭제                          | 선택하면 콘텐츠 유형에 대해 수행된 삭제 작업을 LAR 파일로 내보내거나 가져옵니다.                                                                                                                                                                                                                                                                                      |
| 가져오기 전에 애플리케이션 데이터 삭제(가져오기) | 선택하면 가져오기 프로세스 전에 애플리케이션에서 생성된 모든 데이터가 삭제됩니다. 주의하세요. 다른 애플리케이션이 이 데이터를 참조할 수도 있습니다. 이 프로세스는 취소할 수 없습니다. 확실하지 않은 경우 먼저 내보내기를 수행하세요.                                                                                                                                                                                                   |
| 권한                          | 선택하면 내보내거나 가져온 항목에 할당된 권한이 포함됩니다.                                                                                                                                                                                                                                                                                                    |
| 데이터 업데이트(가져오기)              | 다음 옵션 중 하나를 선택하여 업데이트된 데이터에 어떤 일이 발생하는지 정의합니다. <br/> **미러** : 가져온 LAR 내부의 모든 데이터와 콘텐츠는 소스에 대한 참조를 유지하면서 처음으로 새로 생성됩니다. 동일한 소스에서 후속 가져오기를 수행하면 새 항목을 생성하는 대신 항목이 업데이트됩니다.<br/> **덮어쓰기 미러링** : 미러링과 동일하지만 동일한 이름의 문서나 이미지가 발견되면 덮어쓰기됩니다. <br/> **새로 복사** : 가져온 LAR 내부의 모든 데이터와 콘텐츠는 LAR을 가져올 때마다 현재 사이트와 함께 새 항목으로 생성됩니다. |
| 콘텐츠의 저작권(가져오기)              | 컨텐츠의 저작자를 정의하려면 다음 옵션 중 하나를 선택하십시오. <br/> **원본 저작자를 사용합니다.** : 가능하면 가져온 컨텐츠의 저작자를 유지합니다(원본을 찾을 수 없는 경우 이 옵션은 현재 사용자를 작성자로 사용합니다). <br/> **현재 사용자를 작성자로 사용** : 현재 사용자를 가져온 모든 컨텐츠의 작성자로 지정합니다.                                                                                                                                |

![내보내기/가져오기 프로세스 구성](./exporting-and-importing-style-books/images/03.png)
