# 요소 생성 및 관리

요소는 [Search Blueprints](understanding-search-blueprints.md) 의 기본 빌딩 블록 중 하나이며 Blueprints 애플리케이션 내에서 관리됩니다. 전역 메뉴의 애플리케이션 섹션을 열고 검색 경험 범주를 찾아 검색 청사진을 엽니다. **요소** 탭을 클릭하여 새 요소를 추가하고 기존 요소를 관리합니다.
