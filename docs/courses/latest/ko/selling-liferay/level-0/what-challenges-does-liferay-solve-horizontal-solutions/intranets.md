# 최신 인트라넷

최신 인트라넷에서는 직원들이 하나의 일관된 플랫폼에서 공유하고, 협업하고, 커뮤니케이션할 수 있습니다. 다양한 시스템의 콘텐츠와 기능을 통합하고 집계하는 Liferay DXP의 능력은 개인화된 경험을 제공함으로써 직원의 시간을 절약합니다.

## 대상 고객

최신 인트라넷의 주요 의사 결정자는 HR 책임자 또는 기타 HR 리더입니다. 최신 인트라넷의 상호 연결된 특성으로 인해 IT 리더십도 관련됩니다.

* 크로
* 사장, 이사, 부사장, 관리자

  * HR
  * 일원
  * HR 비즈니스 파트너
* CIO
* 사장, 이사, 부사장, 관리자
  * 그것
  * 이다

## 주요 혜택

최신 인트라넷의 이점은 다음과 같습니다.

* Liferay DXP는 팀이 협업 방식으로 **원격으로 작업할 수 있도록** 지원하는 다양한 도구를 제공합니다.
* 직원마다 역할과 책임이 다르기 때문에 인트라넷은 다운타임을 줄이기 위해 **개인화된 사용자 환경** 을 제공해야 합니다.
* **셀프 서비스 채택을 측정** 하는 것은 어떤 기능과 콘텐츠가 가장 유용한지 파악하고 업데이트해야 하는 콘텐츠나 기능을 식별하는 데 중요합니다.
* 최신 인트라넷은 **플랫폼 문제를 신속하게 해결** 하고 공장이나 사무실이 폐쇄되거나 사용할 수 없는 경우 **비즈니스 연속성** 을 보장하는 데도 도움이 됩니다.
* **보안 액세스 제어** 는 직원이 허용된 콘텐츠 및 기능에만 액세스할 수 있도록 하고 인증 또는 감사 요구 사항을 준수하는 데도 중요합니다.
