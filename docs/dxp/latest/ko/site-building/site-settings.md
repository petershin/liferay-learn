# 사이트 설정

사이트를 [생성했다면](./sites/adding-a-site.md)귀하의 요구와 사용자의 요구를 충족하도록 사용자 정의하고 싶을 것입니다. 원하는 사이트를 만드는 데 도움이 되는 다양한 구성 옵션과 도구가 있습니다. 사이트 메뉴를 통해 이러한 설정에 액세스하여 이름에서 분석 서비스에 이르기까지 사이트의 모든 측면을 구성할 수 있습니다. 사용자에 대한 기본 권한, 사이트에 제공하려는 기본 언어 등을 정의하는 데 도움이 될 수 있습니다.
