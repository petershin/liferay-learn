# 태그 라이브러리
앱, 테마 및 웹 콘텐츠에서 일반적으로 사용되는 UI 구성 요소를 만들기 위한 강력한 taglib 세트에 액세스할 수 있습니다. 이 섹션에서는 다음 taglib를 다룹니다.

* AUI: 양식, 버튼 등과 같은 공통 UI 구성 요소를 생성합니다.

* 차트: 데이터를 시각화합니다. 막대 차트, 선 차트, 분산형 차트, 스플라인 차트 등을 만듭니다.

* Clay: 앱에 대한 경고, 버튼, 드롭다운 메뉴, 양식 요소 등과 같은 [Clay 구성 요소](https://clayui.com/docs/components/alerts.html) 를 만듭니다.

* 프런트엔드: 추가 메뉴, 카드, 관리 표시줄 등과 같이 Portal의 앱 전체에서 일반적으로 사용되는 UI 구성 요소를 생성합니다.

* Liferay UI: 아이콘, 탭 등과 같은 공통 UI 구성 요소를 만듭니다.

* Liferay Util: 추가 리소스 로드, 매개변수 정의, 콘텐츠 버퍼링 등.

```{note} 
Chart taglib를 제외한 각 taglib는 FreeMarker 매크로로 사용할 수 있습니다. Chart taglib는 FreeMarker 매크로로 사용할 수 **없습니다**. 이 섹션의 문서에서는 각 매크로에 사용할 적절한 구문을 제공합니다. 사용 가능한 FreeMarker taglib 매크로의 전체 목록은 [FreeMarker Taglib 매핑 참조](https://help.liferay.com/hc/ko/articles/360029145851-FreeMarker-Taglib-Macros) 를 참조하세요.
```

이 섹션에서는 taglibs를 사용하여 앱을 위한 멋진 사용자 인터페이스를 구축하는 방법을 배웁니다!
